# 🚀 Render Deployment Quick Guide - PG Stay

## One-Minute Summary

Deploy your entire app (Frontend + Backend + PostgreSQL) to Render for **FREE**!

---

## ⚡ Quick Start (3 Steps)

### Step 1: Push to GitHub
```bash
git add .
git commit -m "Ready for Render deployment"
git push origin main
```

### Step 2: Create Render Blueprint
1. Go to https://render.com/dashboard
2. Click **"New +"** → **"Blueprint"**
3. Select your repository
4. Click **"Create Blueprint"**

### Step 3: Set Environment Variables
In Render Dashboard, add these to `pg-stay-backend`:

```
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASSWORD=your_app_password
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```

**That's it! Render will deploy everything automatically! 🎉**

## 🔑 Environment Variables Needed

### From You (@required)
- `EMAIL_USER` - Your Gmail address
- `EMAIL_PASSWORD` - Gmail app password (16 char)
- `RAZORPAY_KEY_ID` - Razorpay merchant key
- `RAZORPAY_KEY_SECRET` - Razorpay secret

### Auto-Generated by Render
- `DATABASE_URL` - PostgreSQL connection string
- `JWT_SECRET` - Random secret for tokens
- `NODE_ENV` - Set to "production"

---

## 📧 Gmail App Password Setup

1. Go to https://myaccount.google.com/apppasswords
2. Select "Mail" + "Windows Computer"
3. Copy 16-character password
4. Paste into `EMAIL_PASSWORD` in Render

**Example:**
```
EMAIL_USER=ravirajh2010@gmail.com
EMAIL_PASSWORD=abcd efgh ijkl mnop
```

3. ✅ Initialize admin user during first backend startup

## ✅ Verify Deployment

1. **Frontend loads?**
   - Visit frontend URL
   - Should see login page

2. **Backend working?**
   - Try login with admin credentials
   - Should connect successfully

3. **Database working?**
   - Add new tenant from admin panel
   - Should save to database

4. **Email working?**
   - Check backend logs
   - Should see email send confirmations

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| "Database connection failed" | Check DATABASE_URL in env vars |
| "Cannot reach frontend" | Clear browser cache, reload |
| "Email not sending" | Verify EMAIL_USER/PASSWORD, check Gmail account |
| "Free tier ran out" | Upgrade to Starter ($7/mo) |
