## 🚀 Ready to Deploy? Follow These Steps:

### Step 1: Commit Your Changes (2 minutes)
```bash
cd "g:\Professional(G)\PG Stay"
git add .
git commit -m "Prepare all files for Render deployment"
git push origin main
```

### Step 2: Create Render Blueprint (5 minutes)
1. Go to https://render.com/dashboard
2. Click **"New +"** → **"Blueprint"**
3. Select your GitHub repository
4. Render will auto-detect `render.yaml`
5. Click **"Create Blueprint"**

### Step 3: Set Environment Variables (5 minutes)
In Render Dashboard, go to `pg-stay-backend` service and add:
```
EMAIL_USER=your_gmail@gmail.com
EMAIL_PASSWORD=your_app_password
RAZORPAY_KEY_ID=your_key
RAZORPAY_KEY_SECRET=your_secret
```

### Step 4: Wait for Deployment (5-10 minutes)
- Render will automatically deploy all services
- Frontend, Backend, and Database working together
- All scheduled jobs active

### Step 5: Verify Deployment (5 minutes)
1. Visit frontend URL
2. Login with admin@pgstay.com / admin123
3. Test adding a tenant
4. Check backend logs for errors
